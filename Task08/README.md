# Task08 - Веб-приложение для управления студентами и экзаменами

## Описание

Веб-приложение для ввода, сохранения в базе данных и вывода информации о составе групп и сданных экзаменах.

## Структура проекта

```
Task08/
├── data/              # База данных SQLite
│   ├── students.db    # Файл базы данных (создается автоматически)
│   └── db_init.sql   # SQL-скрипт инициализации базы данных
├── public/            # Корень веб-сайта
│   ├── index.php      # Главная страница - список студентов
│   ├── student_add.php    # Добавление студента
│   ├── student_edit.php   # Редактирование студента
│   ├── student_delete.php # Удаление студента
│   ├── student_exams.php  # Результаты экзаменов студента
│   ├── exam_add.php       # Добавление результата экзамена
│   ├── exam_edit.php      # Редактирование результата экзамена
│   ├── exam_delete.php    # Удаление результата экзамена
│   ├── config.php         # Конфигурация и подключение к БД
│   └── style.css          # Стили оформления
├── init_db.bat        # Скрипт инициализации базы данных
└── README.md          # Документация
```

## Установка и запуск

### 1. Инициализация базы данных

Запустите скрипт инициализации базы данных:

```bash
init_db.bat
```

Или вручную:

```bash
sqlite3 data\students.db < data\db_init.sql
```

### 2. Запуск веб-сервера

Находясь в каталоге Task08, запустите встроенный PHP веб-сервер:

```bash
php -S localhost:3000 -t public
```

### 3. Открытие приложения

Откройте в браузере:
- http://localhost:3000/index.php
- или просто http://localhost:3000/

## Функциональность

### Главная страница (index.php)

- Список всех студентов с сортировкой по номеру группы и фамилии
- Фильтрация по номеру группы (выпадающий список)
- Для каждой записи:
  - Кнопка "Редактировать" - открывает форму редактирования
  - Кнопка "Удалить" - открывает форму подтверждения удаления
  - Кнопка "Результаты экзаменов" - открывает список экзаменов студента
- Кнопка "Добавить" внизу списка для добавления нового студента

### Управление студентами

- **Добавление студента** (student_add.php):
  - Фамилия, имя, отчество
  - Пол (переключатель: Мужской/Женский)
  - Дата рождения
  - Номер группы (выпадающий список)

- **Редактирование студента** (student_edit.php):
  - Изменение всех полей студента

- **Удаление студента** (student_delete.php):
  - Подтверждение удаления с отображением данных студента

### Управление экзаменами

- **Результаты экзаменов студента** (student_exams.php):
  - Список всех экзаменов студента в хронологическом порядке (от новых к старым)
  - Для каждого экзамена:
    - Кнопка "Редактировать"
    - Кнопка "Удалить"
  - Кнопка "Добавить результат экзамена"

- **Добавление результата экзамена** (exam_add.php):
  - Выбор студента (выпадающий список)
  - Выбор курса обучения на момент сдачи экзамена (1-6 курс)
  - Выбор дисциплины (список фильтруется по направлению студента и курсу)
  - Дата экзамена (можно вводить задним числом)
  - Оценка (2-5)

- **Редактирование результата экзамена** (exam_edit.php):
  - Изменение дисциплины, даты и оценки

- **Удаление результата экзамена** (exam_delete.php):
  - Подтверждение удаления с отображением данных экзамена

## Технические детали

- **База данных**: SQLite
- **PHP**: Используется PDO для работы с базой данных
- **Подготовленные запросы**: Все запросы используют подготовленные запросы для безопасности
- **Альтернативный синтаксис PHP**: Используется шаблонизатор режим без echo
- **Стили**: Современный адаптивный дизайн с градиентным фоном

## Структура базы данных

- **specializations** - направления подготовки
- **groups** - группы студентов (связаны со специализациями)
- **students** - студенты (связаны с группами)
- **subjects** - дисциплины (связаны со специализациями и курсами)
- **exams** - результаты экзаменов (связаны со студентами и дисциплинами)

## Требования

- PHP 7.4 или выше
- SQLite3
- Веб-браузер
